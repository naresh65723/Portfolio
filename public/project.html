Title: Plant Growth Monitoring and Automation Using IoT Platform

Description:

This project focuses on developing an IoT-enabled smart vertical farming system to automate and monitor plant growth conditions. Using sensors, microcontrollers, and cloud integration, the system ensures optimal plant health by regulating water, nutrients, temperature, and humidity in a hydroponic farming setup.

Traditional farming faces challenges such as water scarcity, decreasing cultivable land, and climate dependency. This project leverages IoT and automation to enhance crop yield, reduce manual effort, and promote sustainable agriculture.

Agriculture is a crucial sector that supports the global food supply chain. However, with the increasing world population, cultivable land and water resources are depleting. This calls for innovative farming techniques that can maximize crop yield while minimizing resource consumption.

Hardware Technologies

Microcontrollers & Communication Modules

  • Arduino Uno - Collects sensor data and controls actuators.
  • ESP8266 Wi-Fi Module - Enables cloud communication for remote monitoring.
  • Raspberry Pi - Acts as the central processing unit for automation and control.

Sensors for Real-Time Monitoring

  1. DHT11 Sensor - Measures temperature and humidity.
  2. Capacitive Soil Moisture Sensor - Detects moisture levels in the growing medium.
  3. pH Sensor - Monitors the acidity/alkalinity of the nutrient solution.
  4. TDS (Total Dissolved Solids) Sensor - Measures nutrient concentration in water.

MActuators for Automation

  1. Relay Modules - Controls water pumps, fans, and LED grow lights based on sensor feedback.
  2. Water Pump & Solenoid Valve - Ensures automated water and nutrient supply.
  3. Buzzer Alert System - Notifies users of low water levels or nutrient imbalances.

Software Technologies, Key Features & Future Scope of the Project

Programming & Embedded Software

  1. Arduino IDE (C/C++) - Used for programming Arduino microcontrollers.

Key Features

  • Real-time Monitoring - Tracks temperature, humidity, pH, moisture, and water levels.
  • Automated Control System - Adjusts water, nutrients, and ventilation automatically.
  • IoT-Based Remote Access - Users can monitor plant health from anywhere.
  • Efficient Water & Resource Management - Prevents overuse of water and fertilizers.
  • Improved Crop Growth - Demonstrates a 20% increase in plant weight compared to manual farming.

Future Scope

  1. AI & Machine Learning - Predictive analytics for plant health and growth optimization.
  2. Multi-Crop Adaptability - Extend automation to different types of plants.
  3. Solar Power Integration - Reduce dependency on electrical power.
  4. Mobile App Development - Provide real-time alerts and controls via smartphone.
  5. Scalability for Smart Agriculture - Implement large-scale automated farming.
